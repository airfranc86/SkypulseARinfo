¿Qué es un METAR?

Meteorological Aerodrome Report — reporte rutinario de condiciones actuales

SAEZ 021300Z 18015KT 9999 FEW020 SCT060 BKN100 22/14 Q1018 NOSIG
🏛 Estación ICAO 🕐 Fecha/Hora UTC 💨 Viento 👁 Visibilidad ☁ Nubes 🌡 Temp/Rocío 📏 QNH
Estación
SAEZ

Código ICAO de 4 letras del aeródromo. SAEZ = Ezeiza (Buenos Aires). LEMD = Madrid. KJFK = Nueva York JFK.

Fecha y hora
021300Z

Día del mes (02) + hora (1300) + Z de Zulú (UTC). Siempre en UTC, nunca hora local.

Viento
18015KT

Dirección (180° = Sur) + velocidad (15 nudos). Si hay ráfagas: 18015G28KT. Viento variable: VRB05KT. Calma: 00000KT.

Visibilidad
9999

En metros. 9999 = 10 km o más (visibilidad máxima). 0600 = 600 m. Aeropuertos USA usan millas: 10SM, 1/4SM.

Nubes
FEW020 SCT060 BKN100

Cobertura + altura en centenas de pies.
FEW = 1–2 octas · SCT = 3–4 octas · BKN = 5–7 octas · OVC = 8 octas (cubierto)
020 = 2.000 ft AGL

Temperatura / Punto de rocío
22/14

Temperatura actual (22°C) / Punto de rocío (14°C). Cuando se acercan → niebla inminente. M indica negativo: M03 = −3°C.

QNH
Q1018

Presión al nivel del mar en hPa. Crítico para la altimetría: el piloto ajusta el altímetro a este valor para leer alturas correctas.

Tendencia / Fenómenos
NOSIG · TEMPO · BECMG

NOSIG = sin cambio significativo. TEMPO = cambio temporal. BECMG = cambio gradual hacia X condición. TS = tormenta. FG = niebla.

Categorías de vuelo

Determinadas por techo de nubes y visibilidad — definen qué tipo de vuelo es posible

VFR
Visual Flight Rules
Techo > 3.000 ft
Visib. > 5 km
Vuelo visual sin restricciones
MVFR
Marginal VFR
Techo 1.000–3.000 ft
Visib. 3–5 km
Condiciones límite VFR
IFR
Instrument Flight Rules
Techo 500–1.000 ft
Visib. 1,6–3 km
Solo vuelo instrumental
LIFR
Low IFR
Techo < 500 ft
Visib. < 1,6 km
Condiciones muy severas
